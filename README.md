## CosmoPINNs

This repository contains a PINN framework designed to solve the **canonical differential equations (CDEs)** arising in the computation of **cosmological wavefunction coefficients** under a general power-law FRW universe.

### Overview

Cosmological wavefunction coefficients satisfy differential equations analogous to multiloop Feynman integrals. This project implements a machine-learning based approach to solve the CDE system.
